How it works

  • Click "Add names" to start adding names
  • You can press "Enter/Return" to add a name or click "Add name to list"
  • Click "Finished adding names" when done
  • Click "Pick a random name" to pick a name
  • Click "Edit names on list" to edit the names
  • Click "Reset list" to repopulate the weighted list

The random name picker works by taking the list of names and creating a weighted list that contains each name three times. Each time "Pick a random name" is clicked, Math.random() is used to pick a random name from the weighted list. Everytime a name is picked it is removed from the weighted list so that each name can be chosen up to three times.
